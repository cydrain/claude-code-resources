為什麼要學 Claude Code？

AI 已經不只是聊天工具。Claude Code 讓你直接在電腦上指揮 AI 做事——讀檔案、改程式、跑自動化，像請了一個坐在旁邊的助手。

從「問答」到「做事」

一般 AI 給你答案要你自己做，Claude Code 直接幫你執行。建網頁、整理資料、產報表，講一句話就完成。

非工程師也能用

你不需要懂程式，只要會描述問題。Claude Code 處理技術細節，你專注在「想要什麼結果」就好。

打造你的 AI Agent

設定一次，重複使用。用 Skills 和自動化架構，讓 AI 成為你的個人助理，處理日記、筆記、工作流程。

最強的 AI Coding 工具

在所有 AI 程式助手中，Claude Code 的程式碼品質、上下文理解、多檔案處理能力目前領先。

安全且可控

每個操作都需要你確認才執行，不會自己亂改東西。你是指揮官，AI 是執行者。

This is what we teach
Level 1
Claude 網頁版
你問，它答。
回答完，你自己去做。
  • 對話問答、翻譯、寫文案、腦力激盪
  • 上傳檔案分析（PDF、圖片、Excel）
  • 用 Projects 跨對話保留情境設定
  • 無法在你的電腦上執行任何動作
  • 無法讀寫你硬碟上的檔案
  • 無法操作你電腦裡的其他軟體
  • 無法 24 小時自動跑任務
Level 2
Claude Cowork
住進你的桌面 App，
幫你動手做事。
  • Claude 網頁版所有功能
  • 在指定資料夾內讀取、建立、修改檔案
  • 用 CLAUDE.md + Project 記住偏好和工作流
  • Skills、Plugins 重複工作一句話觸發
  • Connectors 串接多種工具
  • 手機 Dispatch 遠端發任務
  • Subagent 多個分身並行工作
  • Computer Use 操作你電腦裡的其他軟體
  • 限制在沙盒資料夾，無法跨專案協作
  • 不適合 24/7 背景任務
  • 無法寫程式碼打造自己的工具或網頁應用
  • 沒有 Git 版本控制，改壞了無法回滾
Level 3
Claude Code
直接住進你的電腦，
幫你打造自己的 AI 系統。
  • Cowork 所有功能
  • 跨多個資料夾、多個 Git 專案協作
  • 從零寫程式碼，打造網頁工具、Bot、自動化腳本
  • Git 工作流：commit、branch、改壞了一秒回滾
  • 24/7 背景運行（cron + claude -p）
  • Hooks 自訂自動化規則
  • 自訂 slash commands 把整套工作流綁成一句話
  • 部署到 Zeabur、Cloudflare 等雲端服務
  • 你的 AI 助理分身，能代替你做事

Starter Kit 免費・基礎版

透過免費教學文提供給所有人的「開箱包」。把終端機、權限、基本工具一次設定完，告別裝完 Claude Code 卻不知從哪開始的卡關。

  • 01終端機優化（滑鼠、不閃爍、cc 快捷）
  • 02外部編輯器整合（長 Prompt 不再痛苦）
  • 03防手殘三件套（垃圾桶 + 黑名單 + 權限）
  • 04MCP 推薦清單（讀信 / 抓網頁 / 讀檔）
  • 05線上部署懶人包（Zeabur + Cloudflare）
  • 06狀態列（Context / 額度 / Git 一眼看完）

ProKit 設定包（付費解鎖） 迷你課專屬

Starter Kit 幫你把 Claude Code「裝起來」，ProKit 幫你把「AI 分身養起來」。每一包對應一篇迷你課教學文，丟給 Claude 它就會引導你一步步完成。

  • 01AI 分身起始包（讓 AI 一眼讀懂你是誰、怎麼工作）
  • 02第一個專屬 Skill（把重複流程變成一句話指令）
  • 03外部工具整合包（Gmail / 行事曆 / Notion 一次接上）
  • 04AI 規劃模式（動手前先對齊計畫，不走冤枉路）
  • 05 Skill 精華包 （社群圖卡、品牌視覺、日常自動化⋯越拿越多） Coming Soon
